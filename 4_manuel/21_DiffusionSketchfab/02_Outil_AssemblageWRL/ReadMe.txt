marche à suivre
***************
1. Placer tous les *.wrl et les textures liées (*.jpg) dans un dossier LOCAL (disque dur local, C ou D)
2. Copier "ASSEMBLAGE_WRL.bat" dans ce dossier
3. Double-clic sur "ASSEMBLAGE_WRL.bat" pour démarrer l'assemblage des *.wrl présent dans le dossier
4. Une fenêtre DOS s'ouvre et se ferme quelques secondes plus tard.
5. Un nouveau fichier "ASSEMBLAGE.wrl" apparait alors dans le dossier.
		Il contient la fusion de toutes les maquettes *.wrl
